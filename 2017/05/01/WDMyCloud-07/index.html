WDMC 多机型支持 PlexMediaServer 安装

发表于 2017-05-01 更新于 2020-06-02

前言

为了方便更新；以后教程改到博客更新。
也欢迎捐助！！！谢谢。

大名鼎鼎 PlexMediaServer 插件 字幕搜索，影片信息概览功能很强大。

支持机型

  • WDMyCloud （Gen2）
  • WDMyCloudEX2
  • WDMyCloudEX4100
  • WDMyCloudEX2100

安装教程

教程以WDMyCloud为例：

首先确认你的NAS已经安装WDCrack插件，WDMyCloud_WDCrack插件安装方法。
然后下载WDMyCloud_PlexMediaServer_1.1.5.bin(10032016) 链接 密码: r8ar
进WD网页安装插件。
完成。
